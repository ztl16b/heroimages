CSV Format Requirements

The CSV file must contain the following columns:

  • performer_id - Must match the first part of image filenames
  • name_alias - The performer name to display
Example CSV Format
performer_id name_alias other_columns
12345 John Doe ...
67890 Jane Smith ...

Note: Other columns may be present but are not required for this functionality.
